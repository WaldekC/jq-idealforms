Ideal Forms

Ideal Forms is the ultimate framework for building and validating responsive HTML5 forms.

Features:

  • Fully responsive
  • Keyboard support
  • Customizable input types select, radio, checkbox and file
  • Custom datepicker using jQuery UI (with fallback)
  • "On the spot" validation
  • Localization
  • HTML5 placeholder polyfill

Support:

  • Browsers: IE8+, Webkit, Firefox, Opera, iOS5+
  • jQuery: 1.7+, UI 1.8+
  • License: GPL or MIT
